### CompressPDF Tool
## [English]
An automation tool to compress PDF files, designed for handling scanned documents from scanners efficiently.

Key Features:

  - Reduces PDF file size while maintaining document clarity.
  
  - Automates batch processing for multiple files.

Folder Structure:

  - input_folder/: Place your original PDF files here.
  
  - output_folder/: Where compressed files will be saved.
  
  - compresspdf.py: The main script to execute the compression process.

Usage:

  - Put the PDF files into input_folder/.
  
  - Run the script: python compresspdf.py.
  
  - Check the results in output_folder/.

## [Tiếng Việt]
# Công cụ tự động hóa quá trình nén file PDF, phục vụ cho việc xử lý các tài liệu quét (scan) từ máy Scanner một cách hiệu quả.

Tính năng chính:

  - Giảm dung lượng file PDF mà vẫn giữ được độ rõ nét của tài liệu.
  
  - Tự động hóa quy trình xử lý hàng loạt.

Cấu trúc thư mục:

  - input_folder/: Nơi chứa các file PDF gốc cần nén.
  
  - output_folder/: Nơi lưu trữ các file đã được nén thành công.
  
  - compresspdf.py: Script chính thực hiện quy trình nén.

Cách sử dụng:

  - Đặt các file PDF cần nén vào thư mục input_folder/.
  
  - Chạy lệnh: python compresspdf.py.
  
  - Lấy kết quả tại output_folder/.
